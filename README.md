# 🧥 买衣服智能体 - RAG 智能服装推荐系统

一个基于 RAG（Retrieval-Augmented Generation）技术的智能服装推荐问答系统，具有独特的"暴躁"人格设定，能够根据用户的身体特征和偏好提供个性化的服装建议。

## ✨ 功能特性

- **智能服装推荐**：基于用户的身高、体重等特征推荐合适的尺码、颜色和款式
- **RAG 知识检索**：使用向量数据库实现精准的知识检索和问答
- **多轮对话支持**：基于 Redis 的会话历史管理，支持上下文理解
- **知识库管理**：支持上传和管理服装相关的知识文档
- **独特的人格设定**：采用"暴躁"语气回复，增加交互趣味性
- **流式响应**：实时显示 AI 思考过程，提升用户体验
- **防重复上传**：基于 MD5 的文件去重机制

## 🏗️ 技术架构

### 核心技术栈

- **前端框架**：Streamlit - 快速构建 Web 应用
- **大语言模型**：阿里云通义千问 (Qwen3-VL-235B)
- **向量数据库**：Chroma DB - 持久化向量存储
- **嵌入模型**：DashScope text-embedding-v4
- **缓存/会话存储**：Redis - 聊天历史和文件去重
- **框架**：LangChain - LLM 应用开发框架
